Guía Rápida de Integración

Pasos para registrar tu aplicación y usar el iframe

  1. 1
    Crear un Flow – En el dashboard de Truora, construye tu flujo con los validadores (documento + biometría).
  2. 2
    Registrar tu dominio – Comunica a tu integration manager la URL pública donde embedes el iframe (p. ej. en Vercel).
  3. 3
    Crear Account ID – Identifica a tu usuario con un account_id único ([a-zA-Z0-9_.-]+).
  4. 4
    Generar Process Link – Llama a la API para obtener el token de integración y construye la URL: https://identity.truora.com/?token=<api_key>
  5. 5
    Embeder el iframe – Usa el tag con allow="camera", mínimo 450 × 700 px.
  6. 6
    Escuchar postMessage – Captura truora.process.succeeded, truora.process.failed o truora.steps.completed.
  7. 7
    Configurar Webhooks – En el dashboard activa webhooks para recibir notificaciones automáticas de resultados.
